Exercícios de Estruturas de Repetição em JavaScript

Sobre o projeto

Este repositório foi criado para armazenar exercícios de estruturas de repetição desenvolvidos durante meus estudos de programação. Os exercícios foram feitos em JavaScript e têm como objetivo praticar o uso de laços de repetição, contagens e entrada de dados pelo usuário.

Conteúdo

Os exercícios abordam os seguintes temas:

* Contagem crescente utilizando `for`;
* Contagem regressiva utilizando `for`;
* Incrementos personalizados;
* Decrementos personalizados;
* Contagem até um valor informado pelo usuário;
* Verificação de números divisíveis por um valor específico;
* Contagens definidas pelo usuário com valor inicial, final e incremento.

Tecnologias utilizadas

* JavaScript
* Node.js
* Visual Studio Code (VS Code)

 Como executar

1. Clone este repositório:

   bash
   git clone URL_DO_REPOSITORIO
   

2. Acesse a pasta do projeto:

   bash
   cd nome-do-projeto
   

3. Execute o arquivo desejado:

   bash
   node nomeDoArquivo.js
   

Objetivo

O principal objetivo deste projeto é reforçar os conceitos básicos de lógica de programação e estruturas de repetição, fundamentais para o desenvolvimento de software.
